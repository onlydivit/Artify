Artify – Monument Booking System
Artify is an innovative web-based platform built to simplify and enhance the booking process for museums and monuments in Delhi. As a Government-to-Citizen (G2C) solution, it streamlines ticketing, parking, and visitor management, making cultural heritage sites more accessible and enjoyable.

About the Project
Artify modernizes the experience of booking visits to historical monuments. Built using Python and Flask, with SQLAlchemy ORM and a responsive HTML/CSS/JS frontend, it offers solutions for long queues, inefficient scheduling, limited parking, and manual workflows at cultural sites. The project is adaptable for expansion to other cities and sectors, supporting sustainable and tech-driven heritage management.

Features
User registration/login

View monument details, availability, and historical info

Dynamic ticket booking with real-time slot tracking

QR code-based e-tickets and entry verification

Parking slot reservation and management

Secure online payment integration

Admin dashboard for site management, user monitoring, and reporting

Virtual tours (360° views of monuments)

Dynamic pricing and automated student discounts

Multi-language support for broader accessibility

Scalable architecture and optimized for varied devices

Technologies Used
Backend: Python (Flask), SQLAlchemy ORM

Frontend: HTML5, CSS3, JavaScript, Bootstrap

Database: MySQL or SQLite

Additional: QR Code generation (qrcode library), Speech Recognition (speech_recognition), Text-to-Speech (gTTS)

IDE: Visual Studio Code

System Architecture
The user journey includes:

Login/Signup – Secure access to platform features

Home Page – Explore monuments, view availability, and start bookings

Booking Page – Select site, time, number of visitors, and parking

Monument Details – Access historical info, tips, audio guides

Parking Management – Reserve spots in advance

Payment Processing – Choose payment method and confirm booking

Confirmation Page – Receive QR code and booking summary

Usage
Register as a user or log in.

Browse monuments and available slots.

Book tickets, reserve parking, and complete payment.

Receive a QR code for entry.

Explore virtual tours and additional features.

Admin users can manage monuments, bookings, users, and generate reports.
